# Logger App

## Installation
1. Ruby -> ruby 2.5.0p0 (2017-12-25 revision 61468) [x86_64-darwin17]
2. Ruby on Rails -> Rails 5.1.7

### Dependencies:

No Dependencies

### Database
Database is used postgres

### First-Time Setup:

1. Run `bundle install` to install all the gems
2. Run `rails s -p3000` to run the server
